Package Specification

tokens主要类介绍：




Variable表示单个变量


Expression运算表达式，由变量和数字、运算符组成，数字也是运算表达式的一种


Condition原子条件式，由运算表达式和关系符组成


AdvCondition高级条件式，由原子条件式和逻辑运算符组成


EvaluationSentence赋值语句，由变量和运算表达式组成


DecisionSentence条件判断语句，由高级条件表达式组成
